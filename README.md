# 📰 News App

A simple and clean **web-based News Application** built using **HTML,
CSS, and JavaScript**.\
It includes a login page, dynamic news rendering, and a responsive user
interface.

## 🚀 Features

-   🔐 **Login Page** (`login.html`, `login.js`)
-   📰 **Main News Interface** (`index.html`)
-   🎨 **Responsive UI** with dedicated CSS files
-   ⚡ **Dynamic News Rendering** using JavaScript
-   🖼️ **Logo & Favicon Support**
-   🌐 **Pure Frontend App** --- runs on any static hosting service
-   📁 **Clean & simple folder structure**

## 📦 Folder Structure

    News--App/
    |-- favicon.png
    |-- logo.png
    |-- index.html
    |-- login.html

-   **HTML5**
-   **CSS3**

-   Optional: News API integration

## 🧰 Getting Started

### 1️⃣ Clone the Repository

``` bash
git clone https://github.com/shivanshumaurya11/News--App.git
```

### 2️⃣ Navigate to the Project

``` bash
cd News--App
```

### 3️⃣ Run the App

Open **index.html** (or **login.html**) directly in your browser.

## 📝 Usage

-   Start from **login.html** (if authentication logic is implemented)
-   Continue to **index.html**
-   News is loaded and rendered via **script.js**
-   Modify **style.css** or **login.css** to customize UI
